Controls

Default speed: 60 seconds = 1 hour

Time Controls

Fast modes change tick size: 1s/day ticks every 3 hours, 1s/week every 2 days, 1s/month weekly, and 1s/year every 2 months.

Mono Buy / Sell Points

Buying point must be equal to or lower than selling point to avoid arbitrage.

DIV Buy / Sell Points

Grow DIV middle over time

DIV Floor Band

When DIV falls into the floor zone, the treasury desk uses this floor buy/sell band instead of the floating mid-zone band.
Strict static floor: floor buying/selling points cannot be changed.
Strict rising floor: floor can never go down, only up.

DIV Top-Point Rule

Enable top-point lowering when DIV market price is too high.
Top-point = upper trigger. Floor = lower support band. The mid-zone is between the top-point and floor.

Mono-holder DIV Dividend

Dividend Automation

Automatically release DIV dividends to Mono holders until the 365-day dividend goal is reached.
Example: 0.001 DIV per Mono equals 0.10%. The server checks this once per tick and caps the payout so the goal is not intentionally overshot.
